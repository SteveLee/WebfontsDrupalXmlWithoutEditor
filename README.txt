This module makes it easy to use the fonts of webfonts in your drupal website. You need to have your account at webfonts.fonts.com before you begin

.

Installation
:

1) Copy the module folder folder under "drupal\modules" 
2) Login as administrator
3) Go to "Administer"=>"Site building" => "Modules"
4) Activate "Fonts.com webfonts" module from this location
5) Go to "Administer"=>"Site configuration" => "Fonts.com webfonts"
